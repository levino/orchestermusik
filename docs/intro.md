# Orchestermusik

Willkommen bei **Orchestermusik** – einer Sammlung von Noten für Orchestermusiker.

## Über dieses Projekt

Dieses Projekt stellt Noten für Orchestermusik zur Verfügung, die mit [LilyPond](https://lilypond.org/) gesetzt wurden. Alle Noten sind frei verfügbar zum:

- **Teilen** – Geben Sie die Noten an andere Musiker weiter
- **Nutzen** – Verwenden Sie die Noten für Ihre Aufführungen
- **Bearbeiten** – Passen Sie die Noten an Ihre Bedürfnisse an

## LilyPond Quelldateien

Die LilyPond-Quelldateien (`.ly`) sind im Repository verfügbar. Sie können diese Dateien bearbeiten und selbst kompilieren.

## Kompilierte PDFs

Die fertig kompilierten PDF-Noten finden Sie direkt auf dieser Website zum Download.

## Lizenz

Alle Inhalte stehen unter der [CC0 1.0 Universal](https://creativecommons.org/publicdomain/zero/1.0/) Lizenz – Sie können die Werke frei verwenden, ohne um Erlaubnis fragen zu müssen.

## Mitmachen

Möchten Sie Noten beitragen? Besuchen Sie unser [GitHub Repository](https://github.com/levino/orchestermusik) und erstellen Sie einen Pull Request!
