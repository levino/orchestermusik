# Orchestermusik

Eine Sammlung von LilyPond-Noten für Orchestermusik zum Teilen, Nutzen und Bearbeiten.

## Über das Projekt

Dieses Repository enthält LilyPond-Quelldateien und kompilierte PDF-Noten für Orchestermusiker. Die Website wird mit [Docusaurus](https://docusaurus.io/) erstellt.

## Entwicklung

```bash
npm install
npm start
```
